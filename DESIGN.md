\# Brand: NHSM Student Hub

\## Visual Direction

\- \*\*Background\*\*: `#0B0F19` (Deep Navy/Black)

\- \*\*Primary Accent\*\*: Glowing Neon Cyan/Teal

\- \*\*Typography\*\*: Soft light-gray or slate for text. Paragraphs limited to `max-w-3xl` for readability.



\## Spacing \& Structure

\- \*\*Grid\*\*: Strict 8px baseline grid (8, 16, 24, 32, 48, 64px). Never use arbitrary values.

\- \*\*Depth\*\*: No flat borders. Use glassmorphism (`backdrop-blur-md`), multi-layered box shadows, and translucent borders (`border-white/10`).



\## Motion \& Physics

\- \*\*Easing\*\*: All transitions (hover, rotations, scaling) MUST use `transition: all 0.4s cubic-bezier(0.25, 1, 0.5, 1)` to simulate physical momentum. No linear animations.
